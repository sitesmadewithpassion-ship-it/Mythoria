# 🗡️ MYTHORIA - The Official Fantasy Encyclopedia

- **Dark Fantasy Aesthetic**: Black, dark blue, dark purple with gold and cyan accents

- **Parallax Scrolling**: Creates depth and immersion throughout
- **Smooth Animations**: 50+ custom animations and transitions
- **Hero Section**: Rotating world, animated starfield, magical particles

### 📱 Fully Responsive Design
- Mobile-first approach with 100% responsive layouts
- Touch support and swipe gestures for mobile
- Optimized for all screen sizes (320px to 4K)

- **Spell Search & Filtering**: Real-time search across 72+ spells

### 🔧 Technical Excellence
- **SEO Optimized**: Meta tags, structured data, semantic HTML

## 🚀 Quick Start

### For Users
1. **Open in Browser**: Navigate to the website in any modern browser
2. **Explore Sections**: Use the navigation menu to visit different areas
3. **Interact**: Click on kingdoms, spells, characters, and creatures
4. **Search**: Use the spell search to find specific spells
5. **Discover**: Click on the map to discover hidden locations
6. **Learn**: Read timelines and library entries for lore

### For Developers

#### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor or IDE
- Optional: Python or Node.js for local server

#### Installation
```bash
# Clone the repository
git clone https://github.com/sitesmadewithpassion-ship-it/Mythoria.git

# Navigate to project
cd Mythoria

# Option 1: Open directly
open index.html

# Option 2: Use Python local server
python -m http.server 8000

# Option 3: Use Node.js
npx http-server
```

#### File Organization
- **HTML**: All markup in `index.html` (one-file approach for simplicity)
- **CSS**: Organized into three files by purpose
- **JavaScript**: Separated by functionality into three files
- **Data**: All content in `js/data.js` for easy editing

#### Customization
```javascript
// Edit content in js/data.js
const kingdomsData = { /* kingdoms data */ };
const spellsData = [ /* spells array */ ];
const charactersData = [ /* characters array */ ];
const creaturesData = [ /* creatures array */ ];

```

```css
/* Customize colors in styles/main.css */
:root {
    --accent-cyan: #00d9ff;
    --accent-gold: #FFD700;
    --accent-purple: #9933ff;
}
```
